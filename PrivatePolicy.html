Cas spécifique de l’application Call 911.

Dernière mise à jour : 12/08/2024

Introduction

Bienvenue sur l'application "Call 911". Nous prenons très au sérieux la protection de votre vie privée et de vos données. Cette politique de confidentialité décrit les types d'informations que nous collectons, la manière dont nous les utilisons, et les mesures que nous prenons pour les protéger. En utilisant notre application, vous acceptez les pratiques décrites dans cette politique.

Informations collectées

Nous collectons les informations suivantes lorsque vous utilisez l'application "Call 911" :

  • Adresse Gmail : Nous collectons votre adresse Gmail via une connexion sécurisée fournie par Google Cloud.
  • Temps de jeu : Nous suivons le temps que vous passez à jouer dans l'application.
  • Achats : Nous collectons des informations sur les achats effectués en euros et en monnaie numérique $DLZ.
  • Scores : Nous enregistrons vos différents scores obtenus dans l'application afin de les mettre à disposition dans notre leaderboard publiquement accessible.
  • UserName : Nous enregistrons le nom d’utilisateur propre à l’application que vous entrez à votre première connexion dans l’application afin de vous identifier dans le leaderboard et dans le jeu.

Utilisation des informations

Les informations que nous collectons sont utilisées pour :

  • Améliorer votre expérience utilisateur.
  • Suivre et analyser les performances et l'utilisation de l'application.
  • Traiter les paiements pour les achats effectués dans l'application.
  • Vous fournir un support technique et des mises à jour de l'application.

Stockage et sécurité des données

Vos données sont stockées dans une base de données sécurisée protégée par un login et un mot de passe. Nous prenons des mesures de sécurité raisonnables pour protéger vos informations contre tout accès, divulgation, modification ou destruction non autorisés.

Suppression des données

Vous avez le droit de supprimer vos données à tout moment en utilisant le bouton "Delete Profile" dans l'application. Une fois cette action effectuée, vos données seront immédiatement effacées de notre base de données active. Les archives de vos données seront supprimées dans un délai maximum de 30 jours.

Modifications de la politique de confidentialité

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Toute modification sera affichée sur cette page, et, si les changements sont significatifs, nous vous en informerons via l'application.

Contact

Pour toute question concernant cette politique de confidentialité, veuillez nous contacter à l'adresse suivante : contact@thugz-labs.com
